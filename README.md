## Build Docker Images

### Ubuntu 14.04
```
cd ubuntu/14.04

docker build -t fluxcapacitor/package-ubuntu-14.04 .
```

### Java 1.8
```
cd java/1.8

docker build -t fluxcapacitor/package-java-1.8 .
```

### Kafka 0.10
```
cd kafka/0.10

docker build -t fluxcapacitor/package-kafka-0.10 .
```

### Kubernetes 1.3.6
```
cd kubernetes/1.3.6

docker build -t fluxcapacitor/package-kafka-1.3.6 .
```
